# 🟦 Cypress Basics

Resumo direto dos conceitos essenciais de Cypress, uma das ferramentas modernas mais usadas para automação de testes end‑to‑end.

---

## 🧩 O que é Cypress?
- Framework de automação focado em testes **E2E**, **componentes** e **API**.  
- Corre dentro do browser, oferecendo grande visibilidade e debugging.  
- Usa JavaScript e integra facilmente com Node.js.  
- Ideal para aplicações web modernas (SPA).

---

## 🧩 Conceitos Essenciais
- **cy** é o objeto global para comandos (ex.: `cy.get()`, `cy.click()`).  
- Execução automática com **retry** embutido.  
- Suporte nativo a **esperas automáticas** (sem waits manuais).  
- Test Runner interativo para debugging visual.  
- Fixtures para dados estáticos.  
- Interceptações com `cy.intercept()` para mockar APIs.  
- Estrutura recomendada: `e2e/`, `fixtures/`, `support/`.

---

## 🧩 Quando Usar Cypress?
- Projetos front‑end modernos (React, Vue, Angular).  
- Testes rápidos com feedback visual.  
- Necessidade de mocking avançado de API.  
- Equipas que preferem JavaScript/TypeScript.
