# 🟪 Playwright Basics

Resumo direto dos conceitos essenciais de Playwright, framework moderno para automação UI e API, usado em projetos profissionais e na formação CRE.

---

## 🧩 O que é Playwright?
- Framework de automação criado pela Microsoft.  
- Suporta **UI**, **API**, **componentes** e **mobile emuladores**.  
- Multi‑browser: Chromium, Firefox e WebKit.  
- Permite testes rápidos, estáveis e paralelos.  
- Suporta JavaScript, TypeScript, Python, Java e .NET.

---

## 🧩 Conceitos Essenciais
- **Page** é o objeto principal para interações (click, fill, navigate).  
- **Locators** são estáveis e recomendados (`page.getByRole()`, `page.locator()`).  
- Esperas automáticas integradas (auto‑wait).  
- **Fixtures** permitem criar contextos reutilizáveis (ex.: login).  
- **Test runner** nativo com paralelização e reporting.  
- Suporte nativo a **API testing** com `request`.  
- Screenshots, vídeos e tracing para debugging.

---

## 🧩 Quando Usar Playwright?
- Projetos modernos que exigem velocidade e estabilidade.  
- Testes UI + API no mesmo framework.  
- Execução paralela em CI/CD.  
- Equipas que usam JavaScript/TypeScript.
