# 🟨 Selenium Basics

Resumo simples e direto dos conceitos essenciais de Selenium, a ferramenta clássica de automação UI usada no mercado.

---

## 🧩 O que é Selenium?
- Framework open‑source para automação de testes em browsers.  
- Suporta múltiplas linguagens: Java, Python, C#, JavaScript, Ruby.  
- Permite controlar o browser como um utilizador real.  
- Base de muitas entrevistas técnicas e projetos legacy.

---

## 🧩 Componentes Principais
- **Selenium WebDriver** — API que controla o browser.  
- **Browser Drivers** — ChromeDriver, GeckoDriver, EdgeDriver.  
- **Selenium Grid** — execução distribuída e paralela.  
- **Locators** — ID, CSS, XPath, Name, ClassName.

---

## 🧩 Conceitos Essenciais
- Interações: click, type, select, submit, navigate.  
- Esperas: **implicit waits**, **explicit waits**, **WebDriverWait**.  
- Page Object Model (POM) para organização e manutenção.  
- Screenshots e logs para debugging.  
- Execução local, remota ou em pipelines CI/CD.

---

## 🧩 Quando Usar Selenium?
- Projetos enterprise com Java/C#.  
- Ambientes que exigem browsers reais e compatibilidade ampla.  
- Sistemas antigos onde Cypress/Playwright não são opção.
